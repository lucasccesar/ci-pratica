# PIPELINE DE INTEGRAÇÃO CONTÍNUA

## Descrição
Pipeline configurado para executar build e testes automatizados em cada push no GitHub.

## Etapas do pipeline
1. **Checkout:** baixa o código.
2. **Setup Python:** instala Python 3.10.
3. **Instalar dependências:** instala pytest.
4. **Testes:** roda pytest e exibe o resultado.
